Web App

Aplicativo web desenvolvido usando Api PIX disponibilizada pelo Banco Central.
Programado com PHP, JS, MySQL, HTML e CSS

Este projeto consiste na criação de uma plataforma de pagamento destinada à comunidade discente da Universidade Estadual do Maranhão, especialmente voltada para o Centro Acadêmico de Medicina Veterinária. Desenvolvida para gerenciar diversas contribuições, a plataforma visa simplificar operações financeiras, fortalecer o envolvimento dos estudantes na vida acadêmica e garantir a estabilidade financeira do centro acadêmico. Destaca-se pela interface intuitiva e segura, proporcionando uma experiência fluida aos usuários. Em suma, o projeto representa uma contribuição relevante para a gestão financeira do centro acadêmico, promovendo transparência e participação ativa na comunidade discente.

Possuindo um sistema de login ágil.

Com o desejo de ser simples e objetivo para o usuário, mas com uma gestão de dados completa para o administrador.











Ao entrar no aplicativo, os administradores são direcionados até uma pagina única

com o número de usuários ativos, e a contagem das requisições de pagamentos feitas no aplicativo.

Com a opção de "Gerar Extrato", que analisa todas as requisições feitas pela Api e retorna um relatório completo com status de pagamento, data e hora da solicitação e descrição de cada pagamento.

Podendo também corrigir cadastros de outros usuários, caso necessário.

Possui menu para atualização dos dados de cada tipo de contribuição, podendo editar:

  • Valor

  • Descrição gerada para o comprovante de pagamento

  • Tempo de expiração para o pagamento (medido em segundos)


Ao atualizar, os dados são verificados pela Api. Caso hajam alterações indevidas, a criação de novos pagamentos fica suspensa até a correção dos dados.

Na mesma pagina, também é apresentada a soma de todas as requisições feitas no período.
